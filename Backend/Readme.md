Run Control
==========

PHP 7.0.18 | Symfony v3.6 | MySQL | Docker | Docker-compose 

Quick project run
===========

``sudo chmod +x -R docker/unix/``

``sudo ./docker/unix/start_dev_env.sh``

Quick tests run
===========

``sudo ./docker/unix/tests/create_test_environment.sh && ./docker/unix/tests/run_tests.sh``
